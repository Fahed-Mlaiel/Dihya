# Politique de sécurité IT/DevOps

## Description
Politique de gestion des accès pour les applications IT/DevOps.

## Rôles
- admin
- devops
- développeur
- invité

## Règles principales
- Admin : accès total
- DevOps : gestion de ses environnements
- Développeur : accès à ses projets
- Invité : accès limité
