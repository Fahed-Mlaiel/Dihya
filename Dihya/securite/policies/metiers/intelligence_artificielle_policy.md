## Description
Politique de gestion des accès pour les applications IA.

## Rôles

- data scientist
- utilisateur
- invité

## Règles principales
- Admin : accès total
- Data scientist : gestion de ses projets
- Utilisateur : accès à ses projets
- Invité : accès limité
