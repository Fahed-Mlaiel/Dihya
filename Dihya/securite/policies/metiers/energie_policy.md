# Politique de sécurité Énergie

## Description
Politique de gestion des accès pour les applications du secteur énergie.

## Rôles
- admin
- opérateur
- client
- invité

## Règles principales
- Admin : accès total
- Opérateur : gestion de ses sites
- Client : accès à ses données
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
