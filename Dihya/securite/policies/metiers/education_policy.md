# Politique de sécurité Éducation

## Description
Politique de gestion des accès pour les applications éducatives.

## Rôles
- admin
- enseignant
- élève
- invité

## Règles principales
- Admin : accès total
- Enseignant : gestion de ses classes
- Élève : accès à ses cours
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
