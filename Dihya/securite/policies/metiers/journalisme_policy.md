# Politique de sécurité Journalisme

## Description
Politique de gestion des accès pour les applications de journalisme.

## Rôles
- admin
- journaliste
- lecteur
- invité

## Règles principales
- Admin : accès total
- Journaliste : gestion de ses articles
- Lecteur : accès à ses abonnements
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
