# Politique de sécurité Immobilier

## Description
Politique de gestion des accès pour les applications immobilières.

## Rôles
- admin
- agent
- client
- invité

## Règles principales
- Admin : accès total
- Agent : gestion de ses biens
- Client : accès à ses dossiers
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
