# Politique de sécurité Juridique

## Description
Politique de gestion des accès pour les applications juridiques.

## Rôles
- admin
- avocat
- client
- invité

## Règles principales
- Admin : accès total
- Avocat : gestion de ses dossiers
- Client : accès à ses dossiers
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
