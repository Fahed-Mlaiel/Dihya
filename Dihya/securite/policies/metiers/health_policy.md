# Politique de sécurité Santé

## Description
Politique de gestion des accès pour les applications santé.

## Rôles
- admin
- médecin
- patient
- invité

## Règles principales
- Admin : accès total
- Médecin : accès à ses patients
- Patient : accès à ses données
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD, anonymisation
