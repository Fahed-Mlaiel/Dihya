# Politique de sécurité Hôtellerie

## Description
Politique de gestion des accès pour les applications hôtelières.

## Rôles
- admin
- manager
- client
- invité

## Règles principales
- Admin : accès total
- Manager : gestion de ses hôtels
- Client : accès à ses réservations
- Invité : accès limité

## Sécurité
- JWT, CORS, validation, audit, RGPD
