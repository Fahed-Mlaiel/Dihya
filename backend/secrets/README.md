# Dihya Secrets

## Présentation (français)
Ce dossier contient la gestion des secrets, variables d’environnement, et politiques de sécurité pour la plateforme Dihya. Aucun secret réel ne doit être commité. Utilisez `.env.example` pour la structure.

## Overview (English)
This folder contains secret management, environment variables, and security policies for the Dihya platform. Never commit real secrets. Use `.env.example` for structure.

## دليل الأسرار (العربية)
يحتوي هذا المجلد على إدارة الأسرار والمتغيرات البيئية وسياسات الأمان لمنصة ديهيا. لا تقم أبدًا بإضافة أسرار حقيقية.

### Structure
- .env.example : Exemple de structure de variables d’environnement
- GUIDE.md : Guide de gestion des secrets
- POLICY.md : Politique de sécurité

### Sécurité
- Secrets chiffrés, rotation régulière, accès restreint.
- Jamais de secrets en dur dans le code ou les scripts.

### Contribution
Respectez la politique de sécurité et mettez à jour la documentation à chaque changement.
